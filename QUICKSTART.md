# クイックスタートガイド (Quick Start Guide)

## 🚀 すぐに始める (Get Started Immediately)

### 1. アプリケーションを開く (Open the Application)

```bash
# ブラウザで開く (Open in browser)
open index.html
# または (or)
firefox index.html
# または (or)
google-chrome index.html
```

ファイルマネージャーでダブルクリックでも開けます。

### 2. 例のシナリオを確認 (Review Example Scenario)

アプリは以下の現実的な例で自動的にロードされます：

**物件詳細:**
- 物件価格: ¥30,000,000
- 借入額: ¥24,000,000 (LTV 80%)
- 年間賃料: ¥2,400,000 (月額 ¥200,000)
- 表面利回り: 約8%

### 3. 計算実行 (Calculate)

1. 入力値を確認または調整
2. ページ下部の **「計算実行」** ボタンをクリック
3. 結果を確認

### 4. 結果の見方 (Understanding Results)

#### 主要指標 (Key Metrics)
- **表面利回り**: 物件価格に対する年間賃料収入の割合
- **返済比率**: 賃料収入に対するローン返済額の割合（低いほど良い）
- **初年度キャッシュフロー**: 初年度の実際の手残り金額

#### 損益計算書 (Income Statement)
- 会計上の利益を表示
- 減価償却費を含む（非現金費用）
- 税金計算のベース

#### キャッシュフロー計算書 (Cash Flow Statement)
- 実際の現金の出入りを表示
- 元本返済を含む
- 減価償却を含まない
- **投資判断にはこちらが重要**

#### 5年間回収分析 (5-Year Payback)
- 初期投資を5年間で回収できるか判定
- 年2〜5年目の累積C/Fで計算

## 📊 シナリオ管理

### シナリオを保存
1. 上部の「シナリオ名」に名前を入力
2. **「保存」** ボタンをクリック

### シナリオを読込
1. **「読込」** ボタンをクリック
2. リストから選択

### シナリオを比較
1. 2つ以上のシナリオを保存
2. **「比較」** ボタンをクリック
3. 主要指標を並べて確認

## 💾 データ出力

### CSV出力
- **損益計算書をCSVで出力**: 損益計算書のみ
- **キャッシュフロー計算書をCSVで出力**: キャッシュフロー計算書のみ
- **全データをCSVで出力**: 両方のデータ

CSVファイルはExcelやGoogleスプレッドシートで開けます。

## 💡 実用的な使い方

### ケース1: 物件比較
1. 物件Aの情報を入力して「保存」
2. 「新規」ボタンで新しいシナリオ作成
3. 物件Bの情報を入力して「保存」
4. 「比較」ボタンでどちらが有利か確認

### ケース2: 条件の影響を確認
1. ベースシナリオを保存
2. シナリオをコピー（同じ値を入力）
3. 1つの条件だけ変更（例：金利を0.5%上げる）
4. 比較して影響を確認

### ケース3: 返済方法の比較
1. 元利均等返済でシナリオを作成・保存
2. シナリオをコピー
3. 返済方法を「元金均等返済」に変更
4. 比較してどちらが有利か確認

## 🎯 重要なチェックポイント

### 良い投資物件の目安
✅ 表面利回り: 7%以上（都心）、10%以上（地方）
✅ 返済比率: 50%以下が理想、70%以下なら許容範囲
✅ 初年度キャッシュフロー: プラス
✅ 5年間回収: 達成
✅ 10年累積C/F: 大きくプラス

### 危険信号
⚠️ 返済比率 > 85%: 賃料収入のほとんどがローン返済に消える
⚠️ 初年度C/Fがマイナス: 毎月持ち出しが必要
⚠️ 5年間回収未達成: 初期投資の回収に時間がかかりすぎ
⚠️ 累積C/Fが常にマイナス: 長期的に赤字

## 🔧 入力値の調整ガイド

### 現実的な値の範囲

**入居率:**
- 東京都心: 95-98%
- 郊外: 90-95%
- 地方都市: 85-90%

**家賃下落率:**
- 新築・築浅: 0.3-0.5%/年
- 築古: 0.5-1.0%/年
- 良好な立地: 0-0.3%/年

**管理費用:**
- 標準: 賃料の5%
- 自主管理: 賃料の2-3%
- フル委託: 賃料の7-10%

**減価償却年数:**
- 木造: 22年
- 軽量鉄骨: 27年
- 重量鉄骨: 34年
- RC造: 47年

**税率:**
- 個人（給与所得あり）: 20-40%
- 法人: 約30%
- 不動産法人: 約20-25%

## ⚠️ 注意事項

このシミュレーターは教育・参考目的です：

**含まれていない要素:**
- 物件の売却
- 大規模修繕
- 金利変動
- インフレ
- 空室損失の変動
- 突発的な費用

**実際の投資判断には:**
- 税理士・会計士への相談
- 不動産業者の査定
- 金融機関の審査
- 市場調査
- リスク分析

上記を総合的に検討してください。

## 🆘 トラブルシューティング

### 計算結果が表示されない
- ブラウザのJavaScriptが有効か確認
- ブラウザのコンソール（F12）でエラーを確認
- 別のブラウザで試す

### シナリオが保存できない
- ブラウザのlocalStorageが有効か確認
- プライベートブラウジングモードは使わない
- ブラウザのストレージ容量を確認

### CSVがExcelで文字化け
- ファイルは UTF-8 with BOM でエンコード済み
- Excelで「データ」→「テキストファイルから」で開く
- Googleスプレッドシートなら問題なく開ける

### 数値がおかしい
- 入力値を再確認
- 例のシナリオで試す（「新規」ボタン）
- README.mdの計算方法を確認
